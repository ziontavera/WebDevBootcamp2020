jQuery Events

Concepts Learned

  • .click()

    • In Vanilla Javascript, same as adding addEventListener(click, ~~~~)

    $("button").click(function(){ var text = $(this).text(); console.log("You clicked" + text); }); $("button").click(function(){ $(this).css("background", "pink"); });
  • .keypress()

    • adds a keypress listener to events

    different from keydown and keyup, read stackoverflow for specific differences

    $("input[type=text]").keypress(function(event){ if(event.which === 13){ alert("you hit enter"); } });
  • .on()

    • behaves similarly as .addEventListener

    $("button").on("mouseleave", function(){ $(this).css("font-weight", "normal"); }); $("h1").on("click", function(){ $(this).css("color", "purple"); });

    Difference between click() and on(click):

  • .click() only adds listeners for existing elements
  • on() will add listeners for all potential future elements
